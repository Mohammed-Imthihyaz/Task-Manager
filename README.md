# Task Manager

This is a Task Manager Application built wiht spring Boot

## Features
- Create Users
- List All Users
- Create Tasks
- Assign tasks to Different Users
- List all Tasks
- Delete Tasks
- Fetch All Tasks Assigned to a User
- Fetch All Assigned tasks

  ## Requirements

  - java * or higher
  - Maven
  - PostgresSQL
